# 포트번호 
- 구성 
  - 출발지 포트번호 
  - 목적지 포트번호    

0~1024 : 잘알려진 포트 (서버 측 어플리케이션 에서 사용 )
1024 : 예약 되어 있지만 잘 사용되지 않는 포트   
1025 ~ : 랜덤 포트 
- 주로 잘 사용하는 어플리케이션의 번호   
    |어플리케이션|포트 번호|  
    |---|---| 
    |SSH|22|
    |SMTP|25|
    |DNS|53|
    |HTTP|80|
    |POP3|110|
    |HTTPS|443|
> UDP 
효율성 중시 프로토콜 / 데이터를 빠르게 보내는것이 장점 ( 스트리밍 방식의 동영상 서비스와 같은곳에 사용된다.)
