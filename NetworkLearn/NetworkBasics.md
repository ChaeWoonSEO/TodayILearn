# 네트워크 기초 12/23
 **네트워크 의 분류**
> 네트워크는 누가 이용할수 있는 네트워크 인가의 따라 달라지는데 크게 2가지로 분류가 된다.
```
● 사용자를 제한하는 사설 네트워크
● 누구나 이용할수있는 인터넷
```
**사설 네트워크의 한계**
>예를 들어 회사라면 같은 회사안의 직원들만 이용가능하다. 가정 네트워크 라면 그사용자의 가족끼리만 이용가능하다.

**인터넷의 개요**
>인터넷은 다양한 사용자가 이용하는 PC 등을 접속해 자유롭게 전송가능하다. 이런 인터넷은 사용자를 제한하지 않는 네트워크이다.
